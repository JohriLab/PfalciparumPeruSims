# PfalciparumPeruSims
Population genetics forward-in-time simulations of P. falciparum populations in Peru. There are 3 sets of SLiM simulations using different incidence rate assumptions. Selection coefficents may be changed using the SLURM script.

pfalciparum_Peru_timeseries_Maynas.slim - Population size assumptions using Maynas annual incidence rates
SLURM SCRIPT: run_pfalciparum_Peru_timeseries_Maynas.sh

pfalciparum_Peru_timeseries_district.slim - Population size assumptions using Iquitos, Punchana, and San Juan Bautista annual incidence rates
SLURM SCRIPT: run_pfalciparum_Peru_timeseries_district.sh

pfalciparum_Peru_timeseries_full.slim - Population size assumptions using Peru national annual incidence rates
SLURM SCRIPT: run_pfalciparum_Peru_timeseries_full.sh
